# Проверка: отчёт за сегодня и обновление мотивационной ленты

Дата проверки: 2026-05-08

## Что изменено

- AI-проверка отчёта перенесена в рабочий экран дня: вкладка `/focus` отображается в меню как «Сегодня».
- Внутри вкладки «Сегодня» блок называется «Отчёт за сегодня» и показывает только текущий отчёт за сегодняшний день.
- Добавлены понятные статусы отчёта: «Нет отчёта», «Ожидает AI-проверки», «Отчёт принят», «Отчёт отклонён».
- После отклонённого отчёта форма повторной отправки остаётся доступной до конца дня.
- После принятого отчёта показывается сообщение, что штраф за сегодня не будет списан.
- Мотивационная лента теперь получает `refreshSessionId`, `feedVersion`, `generatedAt` и `refreshMessage` от backend.
- Каждая мотивационная карточка получает отдельное поле `displayQuote`.
- При обновлении ленты backend меняет порядок карточек, старается не возвращать последние показанные карточки и обновляет цитаты.
- Frontend использует ключ `image.id + refreshSessionId + feedVersion + index`, чтобы React не сохранял старые карточки визуально.
- После обновления лента прокручивается к первой карточке.

## Проверенные сценарии по коду

- Вкладка AI-проверки не добавлялась в левое меню.
- В левом меню `/focus` переименован в «Сегодня» без изменения route.
- Вкладка «Сегодня» показывает только последний отчёт за текущую дату.
- При `CONFIRMED` форма повторной отправки скрывается.
- При `REJECTED` или `OVERDUE` форма повторной отправки доступна.
- Мотивационная лента получает версию обновления и уникальный ключ карточки на frontend.
- Каждая карточка получает отдельную цитату через `displayQuote`.
- При отсутствии новых изображений backend возвращает мягкое сообщение: «Новые изображения временно не найдены, мы обновили порядок и цитаты.»
